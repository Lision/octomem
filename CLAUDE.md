## Project Overview

Octomem is a universal Agent memory system designed to enhance AI cognition with an "octopus brain" architecture.

**Core Features:**
- Long-term memory
- Content association
- Self-iteration
- Similarity merging
- Contradiction verification
- Truth preservation
- Active exploration

**Architecture:** Multi-agent collaboration with orchestration mechanisms.
